# Destiny Code 수익화 플랜 v1

> 작성일: 2026-06-10
> 전제: 풀이(만세력+AI) + 학습(12챕터 코스+퀴즈+SRS+AI튜터+통변훈련)을 갖춘 현재 기능 기준
> 성격: 설계 문서 — 결제 연동 등 실제 구현은 별도 결정 후 진행

---

## 1. 비용 구조 (왜 수익화가 필요한가)

| 항목 | 현재 | 트래픽 증가 시 |
|---|---|---|
| Gemini API | 무료 등급 (모델당 RPD 약 20회) | Tier 1 결제 필요 — 2.0/2.5 Flash 기준 리포트 1건당 약 1~3원 수준 |
| Render 서버 | 무료 (콜드스타트 최대 1분) | 유료 전환 월 $7~ (콜드스타트 제거) |
| Vercel 프론트 | 무료 (취미 플랜) | 당분간 무료로 충분 |

→ 손익분기 관점: **월 1~2만 원의 고정비**를 감당할 소액 수익만 있어도 무료 한도 제약(하루 AI 호출 수십 회)을 벗어나 안정 운영 가능.

## 2. 무료/유료 구분 원칙

**원칙: 계산은 무료, AI는 유료. 학습 코스는 무료(성장 동력), 고급 훈련은 유료.**

- 토큰이 들지 않는 것(만세력 계산, 명식 분석표, 학습 챕터·퀴즈·복습)은 전부 무료 유지 → 유입·습관 형성 담당
- 토큰이 드는 것(AI 풀이·튜터·통변 채점)은 횟수 제한 + 구독으로 해제

## 3. 요금제 설계 (안)

| | 무료 | 프리미엄 (월 4,900원 안) |
|---|---|---|
| 만세력·명식 분석표 | ✅ 무제한 | ✅ 무제한 |
| 학습 코스 12챕터 + 퀴즈 + 복습(SRS) | ✅ 전체 | ✅ 전체 |
| AI 사주 풀이 | 1일 1회 | 무제한(공정사용 1일 20회) |
| AI 튜터 질문 | 1일 3회 | 무제한(공정사용) |
| 통변 훈련 AI 채점 | 주 1회 | 1일 5회 |
| 신년·궁합 심층 리포트 | ➖ | ✅ |
| 진도 클라우드 동기화(계정) | 코드 백업만 | 자동 동기화 |

- 가격 근거: 국내 운세 앱 구독 시세(포스텔러 등 월 3,900~9,900원) 중간값 이하로 진입 장벽 최소화
- 단건 결제 옵션: 신년 리포트 등 시즌 상품 2,900원 단건 — 구독 거부감 있는 사용자 대응

## 4. 구현 로드맵 (결정 시)

1. **계량 기반 마련 (선행 필수)**: 익명 사용자 식별(브라우저 ID) → AI 호출 횟수 카운트 → 무료 한도 초과 시 안내 모달. *결제 없이도 이것만으로 비용 통제 가능 — 1순위*
2. **계정 시스템**: 이메일 매직링크 또는 카카오 로그인 (Supabase Auth 무료 티어 권장) — 진도·명식 클라우드 저장과 결제 식별 겸용
3. **결제 연동**: 토스페이먼츠(국내 간편) 또는 Stripe(글로벌). PWA 웹 결제라 앱스토어 수수료 없음
4. **Gemini Tier 1 전환 + Render 유료** — 구독자 10명 수준이면 충당

## 5. 리스크 메모

- **콘텐츠 저작권**: 원전 PDF 도표·지식베이스의 원 출처 확인 전에는 유료 상품에 포함하지 않는 것이 안전 (직접 집필한 개념 카드·심화·퀴즈는 문제 없음)
- **운세 서비스 규제**: 점술 서비스는 통신판매업 신고로 충분하나, 환불 정책·면책 고지(오락·참고 목적) 문구 필요
- **무료 한도 악용**: 레이트리밋(완료) + 사용자별 카운트(로드맵 1)로 이중 방어
