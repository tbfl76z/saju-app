## 1. 기획 배경 & 문제 정의

### 1.1 시장 배경
사주·명리 콘텐츠 수요는 꾸준하나, 기존 서비스는 다음 한계를 가진다.

- **블랙박스 해석**: 결과만 보여주고 *왜* 그런지 근거(어떤 글자·십성·격국에서 나왔는지)를 제시하지 않는다.
- **단편적 풀이**: 만세력만 뽑아주거나, 용신 한 가지에 치우친 단조로운 해석에 그친다.
- **고전 점법의 부재**: 자미두수·주역·기문둔갑 등 전통 점법은 별도 유료 프로그램(USB 동글 등)에 갇혀 있고 현대 웹에서 접근하기 어렵다.
- **학습 경로 없음**: 사용자가 사주를 *직접 배우고 해석할* 수 있게 돕는 서비스가 거의 없다.

### 1.2 우리의 접근
Destiny Code는 **"계산–해석–학습–고전점법"을 하나의 흐름으로 통합**한다.

1. **근거 있는 해석** — 모든 단정에 글자/판정 근거를 괄호로 명시("추진력이 강합니다(편관 투출·신강)").
2. **다관점 균형 해석** — 십성·격국·궁위·신살·12운성을 고루 활용(용신 일변도 탈피).
3. **고전 명리 현대화** — 레거시 사주명리 4.0 프로그램(40MB 콘텐츠)을 분석·포팅해 자미두수·주역·기문둔갑까지 웹에서 제공.
4. **배우는 사주** — 11단계 커리큘럼 + 퀴즈 + 간격반복 복습(SRS) + AI 튜터.

## 3. 핵심 가치 제안 (USP)

1. **"왜"를 말하는 AI 해석** — 근거 명시 + 궁위(년·월·일·시)별 인생영역 풀이.
2. **6대 고전 점법의 시각화** — 자미 12궁 명반, 주역 6효 괘상, 기문 9궁 낙서반을 그래픽으로.
3. **검증된 계산** — 점법 산식을 원본 프로그램(오라클)과 대조 검증(자미·기문·주역·명리).
4. **학습 루프 내장** — 보기만 하는 앱이 아니라, 사용자가 *해석가로 성장*하도록 설계.
5. **무가입·무료** — 로컬 저장, AI는 무료 모델 폴백 체인으로 운영 비용 최소화.

## 5. 기능 상세

### 5.1 명식 계산 엔진 (`/calculate`)
- **입력**: 성함(선택)·성별·생년월일(양/음력, 윤달)·시간(시:분 또는 "시간미상")
- **산출**: 네 기둥(년·월·일·시 천간/지지), 오행 분포, 십성(천간·지지), 12운성, 형충회합, 공망, 지장간·투출·통근
- **심층 판정 (`analyze_strength`)** — 해석의 토대:
  - 신강·신약·중화 (세력 채점 + 득령·득지)
  - 격국 (월지 정기 십신 기준 + 투출 강도)
  - 용신·희신·기신 (억부 + 조후 보정: 한겨울 無火/한여름 無水 보강)
- **시간 운세 계산**: 대운 10단계(순역행·대운수), 세운(연운), 월운(월두법), 일운(오늘)

### 5.2 AI 해석 시스템 (`/analyze`, `/analyze/stream`)
- **분석 타입 8종**: 전체운(total)·원국(original)·대운(daeun)·세운(seyun)·월운(wolun)·오늘(today)·궁합(compatibility)·신년(newyear)
  - 신년은 분야별 집중: 연애·재물·직업·건강
- **난이도 2모드**:
  - *전문가*: 십성·12운성·신살·격국 활용, `## 총평 / 정밀 분석 / 개운법 / 대가의 한마디`
  - *쉬운 설명*: 전문용어 최소화, 일상 비유, `## 한눈에 보기 / 요즘 나의 흐름 / 이렇게 해보세요 / 따뜻한 한마디`
- **해석 원칙 (균형)**: 용신·희신·기신은 개운·균형 조언 근거로 한정, 인물 해석은 십성(재·관·인·식·비)·격국·궁위·신살을 고르게 활용
- **RAG 지식베이스**: 명리 매뉴얼·학습자료를 분석타입별 우선순위로 주입(최대 50KB), 출처 흔적 자동 제거
- **모델 폴백 체인**: Gemini 3.5-flash → 2.5-flash → 2.5-flash-lite → 2.0-flash → 2.0-flash-lite → OpenRouter 무료 자동 라우팅 (무료 한도 분산)
- **스트리밍**: SSE 실시간 토큰 출력, 생성 리포트 자동 보관

### 5.3 고전 명리 6모듈 (`/classic/*`)
> 레거시 *사주명리 4.0* 프로그램(USB 동글 잠금)을 역분석해 콘텐츠 DB(40MB, 100+ 테이블, 577만 자)와 산식을 포팅. 모든 점법 산식은 **원본 프로그램(오라클)과 대조 검증**.

| 모듈 | 산출 | 시각화 |
|---|---|---|
| **명리(命理)** | 원명원리·일간론(성격/금전/애정)·일주론·원국종합·십이신살·길흉신살·대운·연운 | 사주 명식표 |
| **자미두수** | 명궁·五行局·14주성 안성 + **12궁 명반**(궁간지·대한 나이) | 4×4 12궁 명반 |
| **주역(周易)** | 동전 6효 작괘 → 본괘 + **변괘(지괘)** | 6효 괘상(양/음/변효) |
| **기문둔갑** | 시가기문 포국(음양둔·삼원·국수, 지반 육의삼기, 천반 직부수동) | **9궁 낙서 방위반** |
| **궁합(宮合)** | 납음궁합·운성궁합·성격궁합 | 점수/관계 |
| **즉석점(卽席占)** | 10개 카테고리(재물·연애·건강·시험 등) 무작위 점사 | 텍스트 |

- **기문 2모드**: 출생국(생년월일시 고정, 평생 명국) / 점단·방위(조회 시점 가변)

### 5.4 학습 시스템 (`/learn/*`)
- **커리큘럼 11챕터**: 음양오행 → 천간 → 지지 → 60갑자 → 지장간 → 십성 → 12운성 → 형충회합 → 신살 → 실전분석 → 통변
- **챕터 학습**: 개념 카드(슬라이드, 도표 포함) → 확인 퀴즈(10문항, seed 재현) → 통과 판정
- **개인화 퀴즈**: 저장한 *내 명식*의 실제 십성·신살로 출제
- **레벨 테스트**: 10문항 진단 → 추천 시작 챕터
- **SRS 복습덱**: SM-2 간격반복, 자기평가(몰랐다/헷갈렸다/알았다) → 다음 복습일 자동 계산
- **통변 훈련**: 명식 보고 직접 해석 작성 → AI 채점(100점) + 30 XP
- **AI 튜터**: 챕터 컨텍스트 기반 질의응답(스트리밍), 틀린 문항 힌트
- **게이미피케이션**: XP·연속 학습일(streak)·완료 챕터·약점 분석
- **진도 백업**: 코드 발급/복원(localStorage ↔ 코드)

### 5.5 저장·공유
- **로컬 프로필**: 최대 30건, "내 명식" 1개 지정(크라운), 불러오기·삭제
- **AI 리포트 보관함**: 생성 리포트 자동 저장(최대 50건, 같은 명식·종류·날짜는 덮어쓰기)
- **공유 링크**: 명식+AI분석을 10자리 코드로 저장(`/shared/[code]` 읽기전용)
- **내보내기**: 명식표·운세 결과를 이미지/PDF로 저장, Web Share API

### 5.6 부가 기능
- **텔레그램 봇**: 대화형 명식 입력·차트·대운·궁합·AI 분석, 웹앱 링크
- **AI 이미지 프롬프트**: 명식 기반 이미지 생성 프롬프트(DALL·E 등)
- **관리자**: 사용량 통계 대시보드(`/admin`, 키 보호) — 일별·경로별·IP별, AI/학습 호출 추적

### 10.1 구현 완료 (운영 중)

- 학습 11챕터·퀴즈·SRS·통변·AI튜터 — ✅
- 저장·공유·PDF·텔레그램 봇 — ✅

### 10.2 검증 이력 (멀티에이전트 적대적 검증)
- **점법 정확성**: 자미(명궁·五行局·14주성·궁간지·대한 順역), 기문(절기 국수표·삼원·포국 9/9), 주역(64괘 round-trip), 명리(오라클 대조) — ✅ 통과

- **AI 해석**: 용신 편중 → 다관점 균형으로 프롬프트 보강

## 11. 운영 · 배포 · 비용

- **배포**: 프론트 Vercel + 백엔드 Render (push to main 자동 배포)
- **AI 비용**: 무료 모델 폴백 체인으로 0원 운영 지향(한도 소진 시 자동 대체)

- **레이트리밋**: AI 튜터·채점 IP당 분당 6회
